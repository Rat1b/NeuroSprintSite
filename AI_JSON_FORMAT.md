# Формат JSON для ИИ — НейроСпринт Планер

Этот документ описывает формат JSON для создания недельного плана с помощью ИИ.

## Пример JSON

```json
{
  "weekStart": "2026-01-12",
  "option": 1,
  "tasks": [
    {
      "day": "ПН",
      "project": "Ф",
      "title": "Прогулка в парке у офиса",
      "duration": 25,
      "startTime": "10:00"
    },
    {
      "day": "ПН",
      "project": "Д",
      "title": "Написать скрипт для видео",
      "duration": 50
    },
    {
      "day": "ПН",
      "project": "К",
      "title": "Сварить вкусный кофе",
      "duration": 15
    },
    {
      "day": "ВТ",
      "project": "Ф",
      "title": "Йога дома",
      "duration": 25,
      "startTime": "09:00"
    },
    {
      "day": "ВТ",
      "project": "Д",
      "title": "Снять видео по скрипту",
      "duration": 50
    },
    {
      "day": "СБ",
      "project": "К",
      "title": "Поход в любимый книжный",
      "duration": 90
    },
    {
      "day": "ВС",
      "project": "Ф",
      "title": "Поплавать в бассейне",
      "duration": 60
    },
    {
      "day": "ВС",
      "project": "Р",
      "title": "Рефлексия по итогам недели",
      "duration": 30
    }
  ]
}
```

## Описание полей

### Корневой объект

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|----------|
| `weekStart` | string | Да | Дата начала недели (понедельник) в формате `YYYY-MM-DD` |
| `option` | number | Нет | Опция структуры недели (1-5), по умолчанию 1 |
| `tasks` | array | Да | Массив задач |

### Объект задачи

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|----------|
| `day` | string | Да | День недели: `ПН`, `ВТ`, `СР`, `ЧТ`, `ПТ`, `СБ`, `ВС` |
| `project` | string | Да | Тип проекта: `Ф`, `Д`, `К`, `Р` |
| `title` | string | Да | Название задачи |
| `duration` | number | Да | Длительность в минутах |
| `startTime` | string | Нет | Время начала в формате `HH:MM` (24-часовой формат) |

## Типы проектов

- **Ф** (Фундамент) — забота о здоровье: прогулки, йога, медитации
- **Д** (Драйв) — развитие и вызовы: обучение, работа над проектами
- **К** (Кайф) — удовольствие: хобби, отдых, приятные занятия
- **Р** (Рефлексия) — анализ: подведение итогов, планирование

## 5 опций структуры недели

### Опция 1 (рекомендуемая) — 90 мин/день = 10ч 30м
- Будни: Ф:25 + Д:50 + К:15
- СБ: К:90
- ВС: Ф:60 + Р:30

### Опция 2 — 45 мин будни + 3ч выходные = 9ч 45м
- Будни: Ф:10 + Д:30 + К:5
- СБ: К:180
- ВС: Ф:150 + Р:30

### Опция 3 — 60 мин будни + 2.5ч выходные = 10ч
- Будни: Ф:10 + Д:45 + К:5
- СБ: К:150
- ВС: Ф:120 + Р:30

### Опция 4 — 2.5ч × 4 дня = 10ч
- ПН, СР, ПТ: Ф:60 + Д:75 + К:15
- ВС: Ф:120 + Р:30

### Опция 5 — 3ч × 3 дня = 9ч
- ПН, СР: Ф:60 + Д:90 + К:30
- ВС: Ф:60 + К:90 + Р:30

## Примеры задач по проектам

### Фундамент (Ф)
- Прогулка на свежем воздухе (25 мин)
- Йога / растяжка дома (25 мин)
- Медитация / дыхательные практики (10-25 мин)
- Дневной сон (25 мин)
- Плавание в бассейне (60 мин)
- Сауна (60 мин)

### Драйв (Д)
- Онлайн-занятие с репетитором (45-50 мин)
- Написать скрипт для видео (50 мин)
- Работа над проектом (50-90 мин)
- Изучение нового навыка (30-50 мин)

### Кайф (К)
- Приготовить и выпить кофе/чай (15 мин)
- Послушать музыку (15 мин)
- Посещение любимого книжного (90 мин)
- Рисование, лепка (90 мин)
- Пикник в парке (90 мин)

### Рефлексия (Р)
- Рефлексия по итогам недели (30 мин)

## Важно для ИИ

1. **Указывай только длительность** (`duration`), не нужно вычислять время окончания
2. **Время начала опционально** — если не указано, задача ставится после предыдущей
3. **Используй русские сокращения** для дней и проектов
4. **Придерживайся рекомендуемых длительностей** для выбранной опции
